# Miya.js
Miya.js is a Collection Runner for Postman.

See also:

  * [Newman](https://github.com/postmanlabs/newman)
  * [MongoDB Node.js Driver](https://docs.mongodb.com/drivers/node/)
  * [node-jq](https://github.com/sanack/node-jq)
